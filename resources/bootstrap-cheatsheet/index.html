Conceptual Colors

  • Primary
  • Success
  • Warning
  • Danger
  • Info
  • Muted

PRIMARY

The primary class is used to show that the element is important. This can also be applied to a background using .bg-primary

SUCCESS

The success class is used to show that the element is successful. This can also be applied to a border using .border .border-success. It is important to first create the border, followed up by the border color class.

WARNING

Warning can be used to notify a user that they are currently missing a requirement. This can be useful for validation feedback to the user with inputs.

DANGER

Danger is often used to show that validation has either failed or that a user is about to do something dangerous, like delete something.

INFO

Info is used to grab the attention and provide information to the user. You can also mix and match different classes as needed.

MUTED

Muted shows something that is not available.
